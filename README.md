# KevinAngular

### INSTALL

- `npm install`
- `ng serve`
- localhost:4200/

#### Dependencias utilizadas

- Front-end: Angular 10
- Back-end: https://mockapi.io/
- Express
- Angular Material
- Flex Layout - Angular
- RxJS
- NgRx
- Ngx-IndexedDB
- Angular Google Maps
- Jasmine JS
- Protractor
- Circle CI

##### Ejecutar pruebas

- Pruebas unitarias: `ng test`
- Pruebas e2e: `ng e2e`
